1.保持初心  !important
&ensp;&ensp;&ensp;&ensp;- 无论怎么样，取得了多高的成就，保持谦卑。
&ensp;&ensp;&ensp;&ensp;- 想明白自己为什么学前端。

3.抓住主要矛盾，把时间精力放在重要的问题上，避免钻牛角尖。
4.科学上网以及善用文档。Search the Friendly Web and Read the Friendly Document ---jyy
